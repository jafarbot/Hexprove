# SEO Monitoring & Review Protocols

## Weekly SEO Review Process

### Step 1: Competitive Landscape Search

Perform web searches for these terms and analyze results:

**Primary Keywords (check weekly):**
```
crypto QA testing company
Web3 QA services
DeFi QA testing
blockchain testing services
```

**Questions to answer:**
- Who is ranking in top 10?
- Any new competitors?
- What content types are ranking? (guides, service pages, listicles)
- Are there featured snippets we could target?
- Any "People Also Ask" questions we should address?

### Step 2: Content Gap Analysis

**Search for long-tail variations:**
```
how to test DeFi app
crypto wallet testing checklist
QA for blockchain startups
Web3 testing best practices
```

**Identify:**
- Topics where no strong content exists
- Questions being asked that we could answer
- Competitor content that's ranking but weak

### Step 3: Industry News & Trends

**Search for recent crypto news:**
```
crypto hack [current month]
DeFi exploit [current month]
Web3 security news
blockchain bug
```

**Look for:**
- Recent incidents that demonstrate need for QA
- Trending topics we can comment on
- Newsjacking opportunities
- Technical developments (new chains, standards, etc.)

### Step 4: HexProve Status Check

**Verify:**
- Is hexprove.com indexed? (search: `site:hexprove.com`)
- How many pages are indexed?
- Any crawl errors or issues mentioned?
- Are blog posts (when live) appearing in search?

## Analytics Interpretation Guide

### Google Search Console Metrics

**Impressions**
- How often HexProve appears in search results
- Growth = more visibility
- Low impressions = need more content or better targeting

**Clicks**
- Actual visits from search
- Click/impression ratio = CTR
- Low CTR = improve titles/descriptions

**Average Position**
- Where HexProve ranks on average
- Position 1-3 = excellent
- Position 4-10 = good, optimize for top 3
- Position 11-20 = close to page 1, prioritize
- Position 20+ = long-term opportunity

**Queries Report**
- What people search to find HexProve
- Identify keywords you're ranking for unexpectedly
- Find keywords close to page 1 to optimize for

### Google Analytics 4 Metrics

**Organic Traffic**
- Users from search engines
- Compare month-over-month, year-over-year
- Segment by landing page

**Engagement**
- Average engagement time
- Bounce rate (engaged sessions / total sessions)
- Pages per session

**Conversions**
- Contact form submissions
- Call bookings
- Newsletter signups

### Interpreting Trends

**Traffic increasing:**
- Content working, create more similar content
- SEO improvements paying off
- Check which pages/keywords driving growth

**Traffic decreasing:**
- Algorithm update? Check industry news
- Competitor outranking? Analyze their content
- Technical issue? Check Search Console for errors
- Seasonal? Compare to previous year

**Rankings dropping:**
- Content outdated? Refresh it
- Competitor published better content? Improve yours
- Lost backlinks? Check backlink profile
- Technical issues? Audit the page

## SEO Action Priority Framework

### Priority 1: Quick Wins (Do This Week)
- Keywords ranking position 4-10 → optimize for top 3
- Thin content pages → expand and improve
- Missing meta descriptions → add them
- Broken links → fix immediately

### Priority 2: Short-term (Do This Month)
- Keywords ranking 11-20 → create supporting content
- Content gaps vs competitors → fill them
- Internal linking improvements
- Schema markup implementation

### Priority 3: Medium-term (This Quarter)
- New content creation for target keywords
- Link building outreach
- Technical SEO improvements
- Content refresh for older posts

### Priority 4: Long-term (Ongoing)
- Brand awareness building
- Thought leadership content
- Industry relationship building
- Authority and trust signals
