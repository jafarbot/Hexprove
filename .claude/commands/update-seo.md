# Update SEO Metadata

Update and optimize SEO metadata across the Hexprove website.

## Key areas to review:

### 1. Metadata in app/layout.tsx
- **Title**: Should include "Hexprove" and key services (QA, Testing, Crypto/Web3)
- **Description**: Compelling 150-160 character summary of Hexprove's services
- **Keywords**: crypto, QA, quality assurance, Web3, blockchain testing, security audit

### 2. Open Graph Tags
- `og:title`: Optimized for social sharing
- `og:description`: Engaging description for social previews
- `og:image`: High-quality preview image (1200x630px recommended)
- `og:url`: Canonical URL (hexprove.io or hexprove.com)
- `og:type`: website

### 3. Twitter Card
- `twitter:card`: summary_large_image
- `twitter:title`: Compelling title for Twitter
- `twitter:description`: Description for Twitter previews
- `twitter:image`: Twitter-optimized image

### 4. Structured Data (JSON-LD)
Consider adding schema.org structured data for:
- Organization
- LocalBusiness (if applicable)
- Service offerings

### 5. Content Optimization
- **Heading hierarchy**: Proper H1, H2, H3 structure
- **Keyword placement**: Natural use of target keywords
- **Internal linking**: Link between services/pages
- **Image alt text**: Descriptive alt text for all images

### 6. Technical SEO
- Canonical URLs
- Robots.txt
- Sitemap.xml
- Mobile responsiveness
- Page load speed
