## Detailed Tracking Plan

### 1. Contact Form Submissions ✅ IMPLEMENTED

**Event Name:** `contact_form_submitted`

**Why It Matters:**
- **Sales:** Direct lead generation metric. Every submission is a potential customer.
- **Marketing:** Measures content effectiveness and conversion funnel.
- **Engineering:** Validates form UX and identifies friction points.

**Data Captured:**
```javascript
{
  source_page: "/",
  has_company: true/false,
  time_on_site: 120, // seconds
  device_type: "mobile/tablet/desktop",
  utm_source: "twitter", // if from marketing campaign
}
```

**Business Value:** 🔥🔥🔥 **CRITICAL** - This is our primary conversion metric.

---

### 2. CTA Click Tracking ✅ IMPLEMENTED

**Event Name:** `cta_click`

**Why It Matters:**
- **Sales:** Measures intent before form submission. High clicks + low submissions = form problem.
- **Marketing:** Tests messaging effectiveness ("Book a Call" vs "Get Started").
- **Engineering:** Validates button placement and mobile responsiveness.

**Data Captured:**
```javascript
{
  cta_location: "hero/navbar/footer",
  cta_text: "Book a Call",
  destination: "/#contact",
  device_type: "mobile/tablet/desktop",
}
```

**Business Value:** 🔥🔥 **HIGH** - Measures top-of-funnel intent.

**Current CTAs to Track:**
- Hero section: "Book a Call", "View Our Work"
- Navbar: "Book a Call"
- Footer: "Get Started"
- Blog posts: "Contact Us" CTAs

---

### 3. Discovery Call Bookings (FUTURE)

**Event Name:** `discovery_call_booked`

**Why It Matters:**
- **Sales:** Most important metric after form submission. Tracks conversion from lead to qualified opportunity.
- **Marketing:** Validates lead quality from different sources.

**Implementation:** When we add Calendly or similar booking tool.

**Data Captured:**
```javascript
{
  booking_source: "contact_form/direct_link",
  time_to_book: 300, // seconds from landing to booking
  utm_campaign: "twitter_launch",
}
```

**Business Value:** 🔥🔥🔥 **CRITICAL** - Direct pipeline metric.

---

### 4. Page Views by Source ✅ IMPLEMENTED

**Event Name:** `page_view`

**Why It Matters:**
- **Marketing:** Core SEO metric. Tracks organic growth, referral traffic, campaign effectiveness.
- **Sales:** Identifies which channels bring qualified leads.
- **Engineering:** Validates site performance and uptime.

**Data Captured:**
```javascript
{
  page_path: "/blog/truebit-26m-logic-bug",
  referrer: "https://twitter.com",
  utm_source: "twitter",
  utm_medium: "social",
  utm_campaign: "launch_week",
  device_type: "mobile",
}
```

**Business Value:** 🔥🔥 **HIGH** - Foundation for all marketing decisions.

**Key Pages to Monitor:**
- Homepage: `/`
- Blog posts: `/blog/*`
- Coming soon page: `/coming-soon` (if active)

---

### 5. Blog Post Engagement (RECOMMENDED)

**Event Name:** `blog_scroll_depth`

**Why It Matters:**
- **Marketing:** Measures content quality. High scroll depth = engaging content = better SEO signals.
- **Sales:** Engaged readers are warmer leads. Track if blog readers convert better.

**Implementation:** Track scroll milestones (25%, 50%, 75%, 100%).

**Data Captured:**
```javascript
{
  post_slug: "truebit-26m-logic-bug",
  scroll_depth: 75, // percentage
  time_on_page: 180, // seconds
  referrer: "twitter",
}
```

**Business Value:** 🔥 **MEDIUM** - Validates content strategy and SEO investment.

---

### 6. Outbound Link Clicks (RECOMMENDED)

**Event Name:** `outbound_link_click`

**Why It Matters:**
- **Marketing:** Tracks which trust signals resonate (Uniswap logo clicks, LinkedIn profile clicks).
- **Sales:** Measures prospect research behavior. Clicking LinkedIn = higher intent.
- **Engineering:** Validates link placement and mobile UX.

**Links to Track:**
- Company logos (Uniswap, OpenSea, Bloomberg, Tradeweb)
- Founder LinkedIn/Twitter profiles
- Portfolio/case study links (when added)

**Data Captured:**
```javascript
{
  link_text: "Uniswap",
  destination: "https://uniswap.org",
  link_location: "hero_experience_section",
  device_type: "desktop",
}
```

**Business Value:** 🔥 **MEDIUM** - Measures trust signal effectiveness.

---

### 7. Mobile vs Desktop Usage ✅ ALREADY TRACKED

**Event Name:** Captured in all events via `device_type`

**Why It Matters:**
- **Engineering:** Prioritizes mobile optimization efforts. If 60% mobile, mobile UX is critical.
- **Marketing:** Informs content format decisions (short-form for mobile, long-form for desktop).
- **Sales:** Validates mobile form UX. High mobile traffic + low mobile conversions = problem.

**Business Value:** 🔥🔥 **HIGH** - Drives UX prioritization.

**Current Status:** Already captured in every event as `device_type: "mobile/tablet/desktop"`.

---

### 8. Navigation Pattern Analysis (RECOMMENDED)

**Event Name:** `section_view`

**Why It Matters:**
- **Marketing:** Identifies which sections resonate. If everyone scrolls to "Why Choose Us", that's a key message.
- **Sales:** Validates sales page structure. If no one reaches pricing, it's buried.
- **Engineering:** Informs layout decisions. Dead sections can be removed or improved.

**Implementation:** Track when sections enter viewport (Intersection Observer).

**Sections to Track:**
- Hero
- What We Catch
- Why Choose Us
- Experience
- Services
- Contact

**Data Captured:**
```javascript
{
  section_name: "why_choose_us",
  time_to_section: 15, // seconds from page load
  scroll_depth_at_view: 40, // percentage
}
```

**Business Value:** 🔥 **MEDIUM** - Optimizes page structure and content hierarchy.

---

### 9. Dark/Light Mode Toggle (FUTURE)

- **Marketing:** User preference data for design decisions.

**Implementation:** When we add theme toggle UI.

**Business Value:** 🔴 **LOW** - Nice-to-have, not business-critical.

## Recommended Umami Setup

### Custom Events to Add

```javascript
// Blog scroll depth
trackEvent('blog_scroll_depth', {
  post_slug: 'truebit-26m-logic-bug',
  scroll_depth: 75,
  time_on_page: 180,
});

// Outbound link clicks
trackEvent('outbound_link_click', {
  link_text: 'Uniswap',
  destination: 'https://uniswap.org',
  link_location: 'hero_experience_section',
});

// Section views
trackEvent('section_view', {
  section_name: 'why_choose_us',
  time_to_section: 15,
});
```

### Umami Dashboard Filters

**For Sales:**
- Filter by event: `contact_form_submitted`, `cta_click`
- Group by: `utm_source`, `device_type`

**For Marketing:**
- Filter by page: `/blog/*`
- Group by: `referrer`, `utm_campaign`

**For Engineering:**
- Filter by: `device_type`
- Group by: `page_path`
